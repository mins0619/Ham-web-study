6과 수업 내용 순서 

0번 전시간에 숙제로 내주었던 마진충돌 현상을 설명한다. 
- main 태그에 디스플레이 인라인 블럭과 너비 100% 를 준다.

1번 메인에 들어갈 디자인을 잡는다.
- 1과에서 이야기 했던 가로로 자르는법 세로로 자르는 법에 대해 이야기 해본다.
- 두번째 네이버는 세로로 두개를 잘랐지만 우리는
그 대신 쉽게 가로 가로 세로로 잘라볼 것이라고 설명한다.
- 먼저 메인 div 태그를 만들고 아이디와 클래스 이름을 준다.

2번 첫번째, 두번째 블럭에 들어갈 코드 디자인을 만든다.
- 가로 순서대로 적어준다.
- 각각의 디브에는 아이디 값을 주어야 한다.
- 이것이 어떤 것을 넣어줄 공간인지 택스트로 표시해준다.

3번 세로로 잘릴 세번째 블럭을 만들어준다.
- 세로로 자를 것 이기 때문에 클래스를 하나 만들어 줄건데
- 디스플레이 인라인 블럭을 만들어 줄거다.
- 각각의 왼쪽과 오른쪽에 들어갈 친구들을 디브안에 디브로 감싸준다. 각각의 아이디 주는거 잊지말자 

4번 css를 건들여보자
- 1. h1~h5사이에 글자 안에있는 글씨의 마진을 0으로 조진다.
- 하다가 공간이 모자라면 블록이 아래로 밀릴 수 있는 점을 설명해주고 직접 너비를 지정하여 해결하는 법을 보여주자 

5번 첫번재 블록의 css 를 디자인한다.
-먼저 첫번째 블록의 마진이 ㅈ같은 문제를 해결해준다. 마진 충돌 해결
- 첫번째 광고와 로그인을 디자인한다.
- 붙이는건 플롯을 써주고 이때 마진은 주지
않고 나중에 붙여준다.

6번 두번째 구역을 디자인한다.
-마찬가지로 플롯 레프트 써주고 마진은 마지막에

7번 --- 매우중요
-이제 새로로 잘라야 하니까 미리 개발자 도구켜서 보여주자
- 이 마진 충돌 현상을 제거 하기위해 우리가 디스플레이를 100%로 주게되면서 첫번재 디브가 두번째 디브에게 공간을 안줄 수 있는데 이때 각각의 넓이를 지정해주면 해결괸다.
-이후 디자인은 알아서~~~

8번 마진을 적어주는데 대충 눈대중으로 해보자
